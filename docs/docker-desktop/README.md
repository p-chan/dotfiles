# Docker Desktop

- Adjust resources
  - Open `Preferences / Resources / ADVANCED`
  - Adjust resources

## Examples

### MacBook Air (Retina, 13-inch, 2020)

- Spec
  - Processor: 1.2 GHz Quad-Core Intel Core i7
  - Memory: 16 GB 3733 MHz LPDDR4
- Docker Resources
  - CPUs: 4
  - Memory: 4.00GB
  - Swap: 1GB
