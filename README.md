# dotfiles

> The dotfiles for fuckin' awesome development environment

## Requirement

- Apple Silicon
- macOS 14 or later

## Install

```sh
DOTFILES_DIR="$HOME/src/github.com/p-chan/dotfiles" bash -c "$(curl -fsSL https://raw.githubusercontent.com/p-chan/dotfiles/main/scripts/install.sh)"
```
