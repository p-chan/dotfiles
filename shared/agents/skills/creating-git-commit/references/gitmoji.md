# gitmoji リファレンス

## フォーマット

```
<intention> [scope?][:?] <message>

[任意の本文]
```

- **intention**: 絵文字（Unicode または `:shortcode:` 形式）
- **scope**: オプション、変更の範囲を示す括弧付きの文字列
- **message**: 変更内容の簡潔な説明

### 絵文字の形式

| 形式                      | メリット                         | デメリット                               |
| ------------------------- | -------------------------------- | ---------------------------------------- |
| Unicode（✨）             | git log の表示が良好、入力が容易 | 一部のターミナル/OSで非対応の可能性      |
| Shortcode（`:sparkles:`） | 全環境で対応                     | プラットフォーム依存のレンダリングが必要 |
